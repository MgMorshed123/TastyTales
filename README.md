# Food Delivery App

A full-stack food delivery application built using React, Node.js, Express, and MongoDB.

## Features

- **User Functionalities:**

  - Add to Cart
  - Checkout
  - Search for Products
  - User Registration and Login

- **Admin Functionalities:**
  - Add New Products
  - Delete Existing Products
  - Manage Orders

## Technologies Used

- **Frontend:** React
- **Backend:** Node.js, Express
- **Database:** MongoDB
